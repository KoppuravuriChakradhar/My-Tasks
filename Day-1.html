What is Browser?

A Browser is a software applicaction that allows you to access and view web pages on the internet. It acts as an interface between you and the server.

Examples: Google Chrome, Mozilla, Firefox.

What is Internet?

The internet is a global network that connects millions of computers and devices, allowing them to share information and communicate with each other.

What is URL?

A URL(Uniform Resource Locator) is the address used to access a specific resource on the internet, such as a website, image, or video.

Example:https://www.google.com

What is Domain?

Domain refers to the address or location of a website or a web resource.

Example:www.amazon.com sub domain: www main domain: amazon Top-level domain: .com

Difference b/w Website and Web App?

Website: A website is a collection of web pages that you can visit through a browser

Example: Google Chrome, Safari,Amazon,Flipcart

Web App: A web app is a more interactive site where you can do things like fill out forms, make purchases, perform tasks. It's like a software application that runs in yours brower

Example: Youtube,Whatsapp,Instagram

Difference between http and https

HTTP is an unsecured protocol used to transfer data over the web

Example: http://example.com

HTTPS is its secure version that encrypts data for safe communication

Example: https://example.com
